## Неделя 1 — Введение в NLP

Лекция 1: Основы обработки естественного языка (NLP)

Лекция знакомит с базовыми понятиями NLP и примерами его применения (анализ тональности, машинный перевод, извлечение информации). Рассматривается стандартный NLP-пайплайн: от сырого текста до модели машинного обучения. Особое внимание уделяется этапам очистки и нормализации текста, включая токенизацию, удаление стоп-слов, стемминг и лемматизацию. Также разбираются методы преобразования текста в векторные признаки (векторизация), такие как Bag-of-Words и TF-IDF.

Семинар демонстрирует практическое применение техник предобработки и анализа текста: знакомство с библиотекой `nltk` для токенизации и удаления шума (пунктуации, стоп-слов). Рассматриваются различия между стеммингом (Snowball) и лемматизацией (Mystem, Pymorphy) для русского и английского языков. В практической части происходит векторизация текстов методами `CountVectorizer` (n-граммы) и `TfidfVectorizer`, после чего обучается модель логистической регрессии для классификации тональности твитов. Также затрагивается тема регулярных выражений для очистки данных.

### Референсы и материалы

* [NLTK (Natural Language Toolkit)](https://www.nltk.org/) - Обработка естественного языка 
* [spaCy](https://spacy.io/) - Industrial-strength NLP 
* [scikit-learn](https://scikit-learn.org/) - Инструменты для ML (CountVectorizer, TfidfVectorizer)
* [Yandex Mystem](https://tech.yandex.ru/mystem/) - Морфологический анализ для русского языка
* [pymorphy3](https://www.google.com/search?q=https://pymorphy3.readthedocs.io/en/latest/) - Морфологический анализатор для Python
* [Snowball](http://snowball.tartarus.org/) - Фреймворк для стемминга
